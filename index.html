How to Update

Edit fires.json on GitHub. Find the department, update the 12-number monthly array, commit.

  1. GitHub repo → fires.json → ✏️ Edit
  2. Ctrl+F → search dept name (e.g. 720 Hempstead)
  3. Update the array — Jan first, Dec last
  4. Commit — site updates in ~60 seconds
